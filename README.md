## Projects

**[Conflictly](https://github.com/Animxh)** — Real-time conflict intelligence platform. FastAPI backend with seven Dockerized services: PostgreSQL/PostGIS, Redis, Elasticsearch, and Celery workers for async processing. Five scrapers (ACLED, GDELT, Reuters RSS, Telegram via Telethon, LiveUAMap), an NLP pipeline using spaCy and sentence-transformers, and a React 18 frontend with a Cesium.js 3D globe. 93 files, beginner-friendly documentation included.

**[NarratorAI](https://github.com/Animxh)** — Mobile-first PWA that combines audio narration with bionic reading. Single-file React app, Firebase backend, packaged for Google Play via Capacitor 6.

**[SQL COVID Analysis](https://github.com/Animxh/SqlProject.git)** — COVID-19 data exploration using SQL: joins, window functions, trend analysis, and data cleaning across a multi-table schema.

**[Tableau COVID Dashboard](https://public.tableau.com/app/profile/animesh.sen8522/viz/CovidDashboard_16795122521650/Dashboard1)** — Interactive Tableau Public dashboard tracking global case trends, mortality rates, and regional comparisons.
